# 🎵 Melodix

A full-stack music streaming platform inspired by Spotify, built using the MERN stack.

## 🚀 Features

- User Authentication (JWT)
- Artist and User Roles
- Upload Songs
- Create Albums
- Play Songs Online

---

## 🛠 Tech Stack

### Backend
- Node.js
- Express.js
- JWT Authentication

### Database
- MongoDB
- Mongoose

### Media Storage
- ImageKit

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Krishan-Das/melodix-fullstack.git
```

### Backend Setup

```bash
cd server
npm install
npm run start
```
